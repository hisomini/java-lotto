## 로또(자동)

- [x] 구입금액에 따라 1000원 단위로 로또를 발급
- [x] 로또는 1에서 45까지의 숫자만 나오도록 제한
- [x] 로또는 6자리의 수로 구성
- [x] 하나의 로또안에서 숫자가 겹치지 않음
- [x] 로또 숫자 오름차순 정렬
- [x] 발행된 로또 출력
- [x] 당첨 번호를 , 기준으로 6개 입력
- [x] 3,4,5,6개 일치한 개수와 총 수익률 출력

## 로또(2등)

- [x] 1~45 범위내에서 보너스 볼 입력
- [x] 보너스 볼을 추가하여 당첨 등급 추가
- [x] [5개 일치, 보너스 볼 일치] 당첨 통계에 추가
